UJIAN TENGAH SEMESTER PRAKTIKUM SIG WEB


Ringkasan Mengenai SIG WEB

Mata Kuliah Praktikum Sistem Informasi Geografi Berbasis web merupakan sebuah aplikasi atau sistem yang digunakan untuk mengumpulkan, mengelola, menganalisis, dan memvisualisasikan data geografis atau spasial melalui internet atau jaringan web. SIG WEB adalah kombinasi antara teknologi informasi dan teknologi geografis yang memungkinkan pengguna untuk menjalankan tugas-tugas berbasis lokasi secara online. Berikut beberapa materi yang telah dipelajari selama praktikum dari minggu ke-1 hingga minggu ke-7.


Minggu ke-1 dan Minggu ke-2 - HTML, CSS, dan WIX

HTML adalah bahasa markup yang digunakan untuk membuat struktur dasar dari sebuah halaman web. HTML sebagai dasar dari hampir seluruh halaman web di internet. HTML digunakan untuk menentukan elemen-elemen dasar dalam halaman web seperti teks, gambar, tautan, daftar, formulir, dan sebagainya.

CSS adalah bahasa yang digunakan untuk mengatur tampilan dan gaya (styling) elemen-elemen yang telah ditentukan dalam HTML. Dengan CSS, pengembang web memungkinkan untuk mengontrol warna, font, tata letak, ukuran, dan elemen desain lainnya pada halaman web. CSS membuat perubahan dalam tampilan halaman web menjadi lebih fleksibel dan mudah dikelola, karena pengguna dapat mengubah styling secara terpusat tanpa harus mengedit setiap elemen di halaman tersebut.

Wix adalah sebuah platform pembuatan situs web yang memungkinkan pengguna untuk membuat situs web sendiri tanpa harus memiliki pengetahuan mendalam tentang pemrograman atau desain web.

Pada pelaksanaan praktikum materi ini, dilakukan dengan membuat sebuah WEB dengan HTML dan CSS sederhana. Selain itu, dilakukan pembuatan Curiculum Vitae dengan menggunakan website WIX.


Minggu ke-3 - JavaScript

JavaScript adalah bahasa pemrograman yang digunakan untuk mengembangkan situs web interaktif dan dinamis. Javascript merupakan salah satu bahasa pemrograman yang paling umum digunakan di seluruh dunia untuk pengembangan web. JavaScript memiliki peran penting dalam menjadikan halaman web lebih responsif, berfungsi, dan interaktif.

Pada praktikum yang telah dilakukan, dilakukan pembuatan script menggunakan java script untuk menjawab lima soal yang telah diberikan.


Minggu ke-4 - Bootstrap Framework

Bootstrap adalah sebuah framework front-end sumber terbuka yang digunakan untuk memudahkan pengembangan situs web dan aplikasi web responsif (responsive web) yang menarik dan berfungsi dengan baik. Framework ini dibuat oleh Twitter dan kemudian dikelola oleh komunitas open source yang luas. Bootstrap menyediakan serangkaian alat, gaya, dan komponen yang siap pakai untuk mempercepat proses pengembangan web.

Pada praktikum yang telah dilakukan, Bootstrap digunakan dalam pembuatan CV yang berisi biodata pribadi pemilik web serta link tugas yang telah dikerjakan selama praktikum SIG WEB


Minggu ke-5 dan Minggu ke-6 - Leaflet JS beserta Pluginnya

Leaflet JS adalah sebuah perpustakaan JavaScript open-source yang digunakan untuk membuat peta interaktif di situs web. Leaflet dirancang untuk menjadi ringan, cepat, mudah digunakan, dan memiliki fokus pada responsif dan kustomisasi. Leaflet merupakan salah satu perpustakaan peta yang paling populer di antara pengembang web dan digunakan secara luas dalam proyek-proyek yang memerlukan integrasi peta interaktif.

Pada praktikum, telah dilakukan pembuatan peta interaktif menggunakan Leaflet JS dengan menambahkan beberapa plugin seperti plugin geolocating dan print
